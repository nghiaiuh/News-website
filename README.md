## Kiбәҝn trГәc CSS

### NguyГӘn tбәҜc

- **Single source of truth**: mб»Қi mГ u, font, spacing, radius, shadow Д‘б»Ғu lГ  CSS variable trong `css/components/root.css`.
- **Module-per-section**: mб»—i section cб»§a trang chб»§ cГі mб»ҷt file CSS riГӘng (`section2.css`, `section3.css`...).
- **Bootstrap utilities** Д‘Ж°б»Јc dГ№ng cho spacing/flex/display, **khГҙng** cho layout phб»©c tбәЎp.
- **KhГҙng hardcode giГЎ trб»Ӣ** вҖ” luГҙn dГ№ng CSS variable hoбә·c `clamp()` Д‘б»ғ responsive.

### Thб»© tб»ұ load CSS (trong `<head>`)

```html

design tokens (:root variables) navbar.css вҶ’ component section1вҖ“4.css вҶ’

```

## Kiбәҝn trГәc JavaScript

### Pattern chГӯnh: Module-per-section + DOMContentLoaded

Mб»—i file JS Д‘Ж°б»Јc `defer` load vГ  khб»ҹi tбәЎo bбәұng `DOMContentLoaded`. KhГҙng cГі global state chung giб»Ҝa cГЎc section JS.

```

sectionThree.js    вҶ’ hover .s3-btn вҶ’ swap .s3-card-gallery img sources
login.js           вҶ’ form validation, fake auth вҶ’ lЖ°u localStorage
dang-ky.js         вҶ’ form validation & Д‘Дғng kГҪ
```


## LЖ°u ГҪ kб»№ thuбәӯt

> **GSAP & scroll-behavior conflict**
>
> Dб»ұ ГЎn ghi Д‘ГЁ `scrollRestoration = "manual"` vГ  `document.documentElement.style.scrollBehavior = "auto"` ngay trong `<head>` (inline script, khГҙng defer). > **KhГҙng xoГЎ hoбә·c defer script nГ y.**

> **Path assets tЖ°ЖЎng Д‘б»‘i**

> TбәҘt cбәЈ `src` vГ  `href` dГ№ng path tЖ°ЖЎng Д‘б»‘i tб»« vб»Ӣ trГӯ file HTML:

> - `../../images/...` вҖ” CSS variable `--sport-icon: url(...)` trong HTML inline style (vГ¬ tГӯnh tб»« vб»Ӣ trГӯ CSS)

> **Self-hosted fonts**
